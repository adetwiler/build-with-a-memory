# Research shelf

Findings that decisions cite. Before a hard-to-reverse call, the options and
tradeoffs get gathered here with sources, so the decision rests on evidence
instead of a hunch, and so a future session can read "did we consider X?" instead
of re-searching it.

## Rules for this shelf

- **Public facts only.** This folder is written as if the repo is already public.
  No strategy, no revenue analysis, no private context. Facts, tradeoffs, and
  links, nothing you would not say out loud.
- **Cite sources.** A finding without a link is an opinion. Label opinions as
  opinions.
- **One question per file.** Keep each findings file scoped to a single decision,
  so the ADR that follows can cite it cleanly.
- **It has to be able to change your mind.** Research written to justify a decision
  already made is theater. If the shelf cannot talk you out of something, it is
  not doing its job.

## What is here

- [landscape-2026.md](landscape-2026.md): the mid-2026 landscape of AI-agent
  memory and multi-agent review. The public prior art this method sits next to,
  and the argument (from the literature) for why some of its choices hold up.
