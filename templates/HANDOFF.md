## Close-out sweeps (run before filing)

- [ ] **Unfinished work:** anything that should have been finished this
      session but was not (a promised change, a verify never run, a decision
      made but never applied)? Finish it or surface it loudly; do not bury it.
- [ ] **Capture misses:** any gotcha, convention, or decision from this
      session not yet routed to its one home (see templates/CAPTURE.md)?
      File the misses now. Usually a no-op if same-pass capture held.
- [ ] **Public drift:** if this session touched public-bound content, did any
      decision here leave a public doc or repo stale? File the gap where it
      will be seen.
- [ ] **Fail closed:** anything headed for a public surface that you are not
      sure is public-safe stays HERE, flagged for the human. Never publish on
      your own judgment.
